Perfect Fit For Me API

Visit our Postman documentation for a much better display: Postman Documentation

This RESTful API allows information concerning our three models to be received.

Cities

api.perfectfitfor.me/cities

Cities by State

api.perfectfitfor.me/cities/state/<state>

Jobs

api.perfectfitfor.me/jobs

Jobs by ID

api.perfectfitfor.me/jobs/id/<id>

Events

api.perfectfitfor.me/events
